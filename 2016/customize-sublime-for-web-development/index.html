为 Web 开发定制 Sublime Text 3

Posted on 2016-04-11 | In 技术

Sublime Text 是非常强大的编辑器，插件功能更是让其拥有无限可能。我用 Sublime 也已经快八个月了，我的 Sublime 也从原来连 Package Control 也没有的纯原版到现在的 Web 开发神器，特此安利一下。

主题推荐

Predawn，就是我上面的截图所使用的主题，也提供 Atom 版。

Flatland，也是扁平风，挺不错的。还有 Monokai 配色版。

插件推荐

Alignment，代码对齐，通常用于一大块变量赋值的时候的等号对齐，强迫症福音；

All Autocompelete，让 Sublime 足以媲美 IDE 的重要插件 —— 项目内代码提示，可以自动补全其他打开的标签页中出现过的函数/变量名

Bracket Highlighter，括号高亮，大堆嵌套条件控制语句的时候就可以不用看得头昏眼花啦

Color Highlighter，CSS 颜色预览，蛮方便的

Compare Side-By-Side，文件比较插件，不多赘述。

ConvertToUTF8，自动转换非 UTF-8 编码至 UTF-8，可配合 EncodingHelper 在状态栏显示当前文件编码。

CSScomb，CSS 规则按照字典序自动排序

DocBlockr，自动生成函数注释块

Emmet，不解释。

File Header，自动生成文件头部，包含作者，创建/修改日期等，模板可自行修改

Git，虽然窝都是 shell 操作的，不过插件版的 Diff 功能还是挺好用的。

IME Support，解决中文输入法不跟随的 BUG，虽然我这并没有效果。。

Markdown Editing，配合 Markdown Preview 就是神器

SideBar Enhancement，侧边栏增强
