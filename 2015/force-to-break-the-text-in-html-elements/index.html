使 HTML 元素中的文字强制自动换行

Posted on 2015-08-09 | In 技术

唔。。有可能你会碰到这种情况：

文本溢出元素了。原因是因为窝用的主题作者没有设置洋文的 break-all，怕截断正常单词，于是就。。 窝以前就直接用 

 标签就算了，然而有些东西放代码高亮里不好看，于是给作者提了个 issue，dalao 说是今晚 commit，但是好像跳票了~


没办法只好自己动手丰衣足食，Google 了一下总结了个临时解决方法：


1.在文章中加个内联 CSS



在文本编辑器中加入就不用窝讲了吧。当然直接加到主题的 style.css 里也可以哦。


记得插入后 **不要 **切换至可视化编辑器，坑爹的 TinyMCE 会把内联 CSS 搞没掉  :(


2.给需要强制换行的 HTML 标签加上 class 属性
按照上面 CSS，就加上 class=”withBreak” ，举个栗子：
